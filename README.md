# Student Mental Health Care Chatbot

## Overview

The Student Mental Health Care Chatbot is a web-based application developed to help students assess their stress levels through a questionnaire. Based on the responses, the system calculates the stress level and provides instant feedback. The application aims to promote mental health awareness among students.

## Features

- Student login using SRM email ID
- CAPTCHA verification
- Stress assessment questionnaire
- Automatic stress level calculation
- Stress percentage analysis
- SQLite database integration
- Simple and user-friendly interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Project Workflow

1. Student logs in using their SRM email.
2. Completes the stress assessment questionnaire.
3. System evaluates the responses.
4. Stress score and percentage are calculated.
5. Stress level is displayed as:
   - Low Stress
   - Moderate Stress
   - High Stress
   - Severe Stress
6. Results are stored in the database.

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Student-Mental-Health-Care-Chatbot.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```text
http://127.0.0.1:5000
```
